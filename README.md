# Buchanan Challenge

Este proyecto está hecho con **Bun**, un entorno de ejecución moderno y rápido para JavaScript, TypeScript y JSX. 

## Instrucciones

1. **Clonar el repositorio**:

   Si aún no tienes el repositorio en tu máquina local, clónalo usando el siguiente comando:
   ```bash
   git clone https://github.com/RoyBuchanandev/Challenge-Buchanan-CreativeDog.git

2. git checkout develop

3. bun install

4. bun dev
